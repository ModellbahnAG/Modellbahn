Platform\+IO ist kein eigenstädniger Editor, sondern ein Plugin, welches für verschiedene Editoren verfügbar ist. Wir nutzen Platform\+IO in Atom.\hypertarget{_platform_i_o_newproject}{}\doxysection{Ein neues Projekt erstellen}\label{_platform_i_o_newproject}
Anders als bei der Arduino I\+DE können wir nicht einfach Platform\+IO öffnen und direkt drauf los programmieren, sondern müssen zuerst ein neues Projekt erstellen. /n Dafür müssen wir im \{{\ttfamily Home} Tab\} von Platform\+IO \{{\ttfamily New} Project\} wählen und das neue Projekt konfigurieren. Als Name sollte ein aussagekräftiger Name gewählt werden, damit das Pfojekt auch später wieder gefunden wird. Als Board wird der Typ des Arduinos angegeben, in den meisten Fällen ist das der \{{\ttfamily Arduino} Uno\}. Dies lässt sich später noch in der platformioini \{{\ttfamily platformio.\+ini\}} Datei anpassen. Als Framework M\+U\+SS Arduino ausgewählt sein. Um Github nutzen zu können, muss das Projekt im Ordner {\ttfamily Modellbahn} oder in einem seiner Unterordner gespeichert sein. /n Sobald das Projekt erstellt wurde, kann in der sich im {\ttfamily src} Ordner befindenden \{{\ttfamily main.\+cpp\}} Datei das Programm geschrieben werden.
